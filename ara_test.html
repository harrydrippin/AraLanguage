사용법

간단한 사용법을 설명합니다. 자세한 사용법은 여기를 참조하세요.

요소

사용할 수 있는 요소들을 설명합니다.<br><br> 아래는 구현되었으며, 모든 사용이 가능한 요소들입니다:

  • 숫자
  • 참과 거짓
  • 문자 (혹은 문자열)<br><br>

아래는 구현되었지만, 정상적으로 연산이 불가능한 상태인 요소들입니다:

  • 배열
  • 사전 (Dictionary)
  • 쌍 (튜플, Tuple)

Python에서 사용할 수 있는 요소들을 모두 사용할 수 있습니다만, 아직 정상적인 연산을 지원하지 않는 것들이 있습니다. 업데이트 예정이니 양해 부탁드립니다.

먼저 숫자입니다. 숫자는 아시다시피 1, 39, 137... 등의 값들을 말하며, 3.141592나 148.13783과 같은 소수점을 포함한 값도 역시 계산이 가능합니다. 계산은 사칙연산이 기본적으로 가능하며, 제곱 등의 더 발전한 계산들도 할 수 있습니다. 방법은 뒤에 설명합니다.

다음은 참과 거짓입니다. 불(혹은 부울) 대수로 알려져 있는 값입니다. 영어로 true, false라고 하며, 어떤 논리가 맞는가, 혹은 맞지 않는가를 판별하기 위한 값입니다. 아라에서는 참, 거짓으로 사용합니다. 이 경우, 계산은 또는 혹은 그리고 등의 논리 계산을 사용합니다. 방법은 역시 뒤에 설명합니다.

다음은 문자입니다. '가'와 같이 하나의 문자를 사용할 수도 있고, "이것은 문장입니다."와 같이 띄어쓰기를 포함한 여러 개의 문자열로 만들어 사용할 수 있습니다. 문자열은 합치고 자르는 등의 연산이 존재합니다.

다음은 배열입니다. 리스트라고도 하는데, 배열이란, 여러가지 값들을 순서가 있게 나열하여 다루기 쉽게 하나로 합쳐놓은 값입니다. [1, 4, 5]와 같이 사용할 수 있습니다. 배열 안에는 숫자 뿐만 아니라, 문자열이나 다른 값들도 들어갈 수 있습니다. 또, [3, 7, [5, 4, 9]]와 같이 배열 안에 또 다른 배열을 넣을 수 있습니다.

사전과 쌍은 현재 선언이 가능하지만, 의미가 없으므로 구현 완료 후 설명이 탑재됩니다.

연산자

연산자는, 간단히 말해서, 사칙연산이라고 생각하시면 됩니다. +, -, *, /와 같은 것들이 존재합니다. 순서대로, 더하기, 빼기, 곱하기, 나누기입니다. 1 + 3이나, 5 * 7과 같이 평소 사용하시는 것처럼 쓰시면 됩니다. 이것들 외에도 %, **, 또 다른 여러가지가 있으나, 많이 사용하지 않습니다. 자세한 설명은 상단의 링크를 참조하세요.

문법

아라에서 행해지는 모든 명령은 같은 형식을 취합니다:

[명사](조사) [명사](조사) [서술어]

다시 말해서, 명사와 조사로 이루어진 조합은 인자, 서술어는 명령어의 역할을 하는 것입니다.<br> 예는 다음과 같습니다:

a에 10을 넣기
a를 보여주기

위의 예에서, a는 명사, 에나 을/를은 조사, 넣기와 출력하기는 서술어입니다. 자세한 정의는 각 링크를 참조하세요.

명령어

현재 사용한 명령어에는 다음과 같은 종류가 있습니다.

  • 보여주기 (출력하기)
  • 넣기
  • 더하기/빼기/곱하기/나누기<br> 이것들 이외에도 다양한 명령어가 존재할 수 있으나, 현재로써는 간단히 이것들만 설명합니다. 다음은 위 명령어들의 설명입니다.<br><br>

보여주기는 조사 을/를을 인자로 가지는 명령어입니다. 인자로 받은 값을 화면에 출력합니다. 예는 다음과 같습니다.

a를 보여주기
"안녕하세요!"를 보여주기
3.141592를 보여주기

<br><br> 넣기는 조사 에, 을/를을 인자로 가지는 명령어입니다. 에인자로 받은 변수명에 을/를로 받은 값을 넣습니다. 예는 다음과 같습니다.

a에 10을 넣기
첫째에 30을 넣기
둘째에 [1, 4, 7]을 넣기
결과에 "성공했습니다!"를 넣기

<br><br> 더하기... 는 조사 에/에서, 을/를을 인자로 가지는 명령어입니다. 연산자의 역할을 그대로 대신합니다. 예는 다음과 같습니다.

a에 10을 더하기   // 이것은 "a에 a + 10을 넣기"와 같습니다.
a에서 10을 빼기   // 이것은 "a에 a - 10을 넣기"와 같습니다.
a에 5를 곱하기    // 이것은 "a에 a * 5을 넣기"와 같습니다.
a에서 9를 나누기  // 이것은 "a에 a / 9을 넣기"와 같습니다.
